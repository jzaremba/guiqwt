Welcome to guiqwt's documentation!
===================================

.. image:: images/guiqwt.png
   :align: center

Based on PyQwt (plotting widgets for PyQt4 graphical user interfaces) and 
on the scientific modules NumPy and SciPy, `guiqwt` is a Python library 
providing efficient 2D data-plotting features (curve/image visualization 
and related tools) for interactive computing and signal/image processing 
application development.

External resources:

* Downloads, bug reports and feature requests: `SourceForge`_
* Help, support and discussions around the project: `Google Group`_
  
.. _SourceForge: http://sourceforge.net/projects/guiqwt/
.. _Google Group: http://groups.google.fr/group/guidata_guiqwt
