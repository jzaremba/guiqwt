Welcome to guiqwt's documentation!
===================================

.. image:: images/guiqwt.png
   :align: center

Based on PyQwt (plot widgets for PyQt4 graphical user interfaces) and on 
scientific modules NumPy and SciPy, `guiqwt` is a Python library for efficient 
2-D data plotting (curves, 1-D and 2-D histograms, images) and signal/image 
processing application development.

External resources:

* Downloads, bug reports and feature requests: `SourceForge`_
* Help, support and discussions around the project: `Google Group`_
  
.. _SourceForge: http://sourceforge.net/projects/guiqwt/
.. _Google Group: http://groups.google.fr/group/guidata_guiqwt
